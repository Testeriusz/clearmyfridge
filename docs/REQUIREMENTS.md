# ClearMyFridge — User Stories

## Fridge Tracker

**US-01**
As a user, I want to add items to my fridge with a name, quantity, and expiry date, so that I always have an accurate picture of what I have at home.

**US-02**
As a user, I want to edit or remove items from my fridge inventory, so that my list stays up to date as I use or discard food.

---

## Expiry Alerts

**US-03**
As a user, I want to receive a push notification when an item is expiring within 3 days, so that I can prioritise using it before it goes to waste.

**US-04**
As a user, I want to see a colour-coded expiry status on each fridge item (green / amber / red), so that I can spot items that need attention at a glance.

---

## AI Recipes with Macros

**US-05**
As a user, I want the app to suggest recipes based on the ingredients currently in my fridge, so that I can cook a meal without needing to buy anything extra.

**US-06**
As a user, I want each suggested recipe to display calorie and macro information (protein, carbs, fat), so that I can make food choices that align with my nutritional goals.

---

## Shopping List

**US-07**
As a user, I want to add items to a shopping list directly from a recipe I'm missing ingredients for, so that I don't have to manually cross-reference what I need to buy.

**US-08**
As a user, I want to check off items on my shopping list as I shop and have them automatically added to my fridge inventory, so that updating my fridge requires no extra steps.

---

## Barcode Scan

**US-09**
As a user, I want to scan a product's barcode to add it to my fridge, so that I can log groceries quickly without typing out names and details manually.

**US-10**
As a user, I want the barcode scanner to auto-fill the product name, quantity, and default expiry period, so that adding a full weekly shop takes seconds rather than minutes.

---

## Dietary Filters

**US-11**
As a user, I want to set dietary preferences (e.g. vegetarian, gluten-free, dairy-free), so that recipe suggestions only show meals that are safe and suitable for me.

**US-12**
As a user, I want to filter the recipe list by dietary category at any time, so that I can quickly find options that match my mood or specific nutritional needs on a given day.
